# **How to Play**
Run Image_Oscillation.pde on Processing 4
<br>
Processing上で**Image_Oscillation.pde**を実行してください。

← / → Move character

↑ / ↓ Change frequency (pitch change)

LMB Start / Ending screen transition

キーボード十字キー　← / →　キャラクターの左右移動

↑ / ↓　周波数の変更（ピッチ変化）

左マウスクリック　スタート画面、エンディング画面　遷移



# **Required**
・Processing 4.3
